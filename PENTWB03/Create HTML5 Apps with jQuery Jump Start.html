Tipos de selectores

Selectores Básicos

Selectores de atributos

$( "selector[attr]" );

Manipulando el DOM

Manipular elementos

$("content").appendTo(" selector "); -- Nos permite tomar un elemento y moverlo a algún otro lugar en el DOM de nuestra página.
Parrafo donde agregaremos el texto


$("content").prependTo(" selector "); -- Nos permite agregar elementos después del elemento seleccionado
$("selector").clone().appendTo("selector2");

Modificando Atributos

$("selector").attr("name","value");
$("selector").prop("name","value");
$("selector").removeAttr("name");
$("selector").removeProp("prop");
$("selector").attr("name", function(attr) { ... } );

Elementos con estilo CSS

$("selector").css("style","value");
$("selector").css({"style":"value","style":"value"});
$("selector").css("style", function(index, value) { ... } );
