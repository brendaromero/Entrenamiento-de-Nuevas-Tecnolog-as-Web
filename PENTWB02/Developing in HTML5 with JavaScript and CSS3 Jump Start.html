Selectores de CSS3 y propiedades de estilo

Selector con clase

.intro { ... }

Selecciona todos los elementos el nombre de la clase

Selector con id

#myId{...}

Selecciona todos los elementos con el nombre id

Selector *

* {... }

Selecciona todos los elementos

Selector de elementos

h4 {... }

Selecciona el elemento por su nombre, se puede nombrar uno o mas elementos (h4, div...)

Selector elemento > elemento

div > p {... }

Selecciona el elemento por su nombre cuando tiene un padre

Selector con atributos

[title~=valor]{...}

Selecciona todos los elementos con un atributo de título que contiene la palabra que se menciona

[lang|=valor] {...}

Selecciona todos los elementos con un valor de atributo lang que comienza con el valor indicado

a[href^=valor] {...}

Selecciona cada elemento cuyo valor de atributo href comienza con el valor indicado

a[href$=".valor"] {...}

Selecciona cada elemento cuyo valor de atributo href finaliza con el valor indicado"

a[href*="valor"] {...}

Selecciona cada elemento cuyo valor de atributo href contiene la subcadena del valor indicado

Objeto promesa

Una Promesa se encuentra en uno de los siguientes estados:

pendiente (pending): estado inicial, no cumplida o rechazada.
cumplida (fulfilled): significa que la operación se completó satisfactoriamente.
rechazada (rejected): significa que la operación falló.
