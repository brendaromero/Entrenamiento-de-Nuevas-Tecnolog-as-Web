Introducción con CSS

Sintaxis:

selector{
-----------------------> ¿A qué elementos o tipos de elementos desea aplicar el estilo?
property: value
-----------------------> ¿Cómo quieres que se vean y dónde quieres que aparezcan?
}


Selección de elementos básicos

-Por Elemento

Selecciona el elemento por su nombre, se puede nombrar uno o mas elementos (h4, div...)
h2 {
font-family: verdana, sans-serif;
font-weight: bold;
color: blue;
}

-Por Clase

Se aplica a todos los elementos con una clase coincidente
.title {
color:red;
}
Por defecto, las clases están disponibles para todos los elementos
Límite a elementos específicos por clase precedente con el nombre del elemento
div.title {
color:red;
}

-Por Id

Se aplica al elemento por su ID
#header {
font-family: verdana, sans-serif;
font-weight: bold;
color: blue;
}


Aplicación de CSS a páginas y elementos

-Enlace a un archivo externo

< link rel='stylesheet' type='text/css' href='site.css' />

-Añadir la sección < style> a la página

< style>
/* sintaxis de CSS */
.

-Use el atributo de estilo

< div style='property:value;property:value;'>Hola! .
Herencia CSS

-Conflictos

Los conflictos solo surgen cuando la misma propiedad tiene dos valores diferentes

-Aplicación de estilo

Lo más cercano al elemento tiene prioridad

-Conflictos de elementos

Se pueden aplicar múltiples selectores a un elemento
Una de las características principales de CSS es la herencia de los estilos definidos para los elementos. Cuando se establece el valor de una propiedad CSS en un elemento, sus elementos descendientes heredan de forma automática el valor de esa propiedad.

Pseudo-clases

Las pseudo-clases se basan en el estado de la página

Las Pseudo-classes, junto con los pseudo-elements, permiten aplicar un estilo a un elemento no sólo en relación con el contenido de la estructura del documento.

- Ubicación de un elemento

- Estado de un elemento

Sintaxis

selector:pseudo-class {
property: value;
}
